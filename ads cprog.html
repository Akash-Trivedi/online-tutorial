C Programming

History:

The C programming language was devised in the early 1970s by Dennis M. Ritchie an employee from Bell Labs (AT&T). In the 1960s Ritchie worked, with several other employees of Bell Labs (AT&T), on a project called Multics.The origin of C is closely tied to the development of the Unix operating system, originally implemented in assembly language on a PDP-7 by Dennis Ritchie and Ken Thompson, incorporating several ideas from colleagues. ... Unix was one of the first operating system kernels implemented in a language other than assembly.
